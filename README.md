使用github仓库保存并更新下自己常用的一些脚本，方便修改和备份。

- R_note  
集百家之所长，学习各个地方挖来的教程做的一些测试笔记
- py_note  
python的学习笔记
- scripts  
关于组学的一些上游分析的代码，更多的请参考[yexianingyue](https://github.com/yexianingyue)和[RChGO](https://github.com/RChGO)两位大佬的笔记
